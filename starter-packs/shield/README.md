## Getting Started

Your Shield agent is pre-configured and ready to protect. Here's what you can do:

```bash
# Audit a repository
python my_agent/shim.py --action audit --path /path/to/repo

# Validate configuration
python my_agent/shim.py --action validate --config config.yaml
```

## Customization

- **Strictness**: Lenient, Standard, Strict
- **Scope**: Code, Configs, Dependencies
- **Reporting**: Verbose, Summary, Critical-Only
